## Premiere connexion

Apres la creation de votre compte, connectez-vous a l'application Kouroukan. Un assistant de configuration vous guide pour les etapes initiales.

## Informations de l'etablissement

Renseignez les informations generales :

- **Nom de l'etablissement** : le nom officiel
- **Type** : public, prive laique, prive franco-arabe (medersa), communautaire ou confessionnel
- **Cycles** : prescolaire, primaire (CP1-CM2), college (7eme-10eme), lycee (11eme-Terminale)
- **Adresse et region** : localisation precise

## Configuration des niveaux et classes

### Pour le primaire

Les niveaux suivent la nomenclature guineenne :
- CP1, CP2, CE1, CE2, CM1, CM2

### Pour le college

- 7eme annee, 8eme annee, 9eme annee, 10eme annee

### Pour le lycee

- 11eme annee, 12eme annee, Terminale (series SE, SM, SS)

Pour chaque niveau, creez vos classes (ex : CM2-A, CM2-B) et indiquez la capacite.

## Annee scolaire

Definissez l'annee scolaire en cours avec les dates de debut et de fin, ainsi que les trimestres ou semestres selon votre cycle.

## Parametrage des frais

Si vous utilisez le module Finances, configurez :

- Les types de frais (inscription, scolarite, cantine, uniforme, etc.)
- Les montants par niveau
- Les echeanciers de paiement
- L'integration Mobile Money (Orange Money, Soutra Money, MTN MoMo)

## Ajout du personnel

Invitez les membres de votre equipe :

- **Directeurs adjoints** : acces complet a la gestion
- **Enseignants** : acces aux notes, presences et emplois du temps
- **Comptables** : acces au module finances
- **Secretaires** : acces aux inscriptions et documents

Chaque personne recoit un SMS avec ses identifiants de connexion.

## Prochaine etape

Votre etablissement est configure. Vous pouvez maintenant commencer les inscriptions. Consultez le guide [Module Inscriptions](/aide/utilisation/inscriptions).
